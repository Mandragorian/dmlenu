Module Ordering

module Ordering: sig .. end
Functions used to (re)order the list of candidates after matching.

State transition functions (defined in State, used by Dmlenu.run) use the default Ordering.reorder_matched reordering function, that you can change using Ordering.set_reorder_matched_fun.
type matched_candidate = Candidate.t * Matching.result
